# My New Website

*June 18, 2026*

---

I've been wanting to revamp the site from my original HTML, CSS, and Javascript version for quite some time. After working with Vue, Vuetify, and Tailwind for work I decided it was a great time to upgrade! Also, with the advancements of AI, I knew it would be a great time to experiment with Claude on my own, while also getting something done I had been hoping to do!

## Why I Rebuilt My Site

This site originally used HTML, CSS, and Javascript. This worked, and was honestly easier for me to personally maintain at the time since that's what most of my front end development experience was in. However, working with Vue made me excited to convert to a more modern option. It was also a good way to learn a bit more about the framework since we use at my work (although we are still on Vue 2 for now).

The site also used to be much more rudimentary. It was the same generic concepts, but I am by no means a fantastic UI designer, artist, or CSS wizard. However, Claude is pretty dang good at it! It was truly awesome to make my vision come to life in minutes instead of hours or days. It also finally got me to a point where I was able to add a simple blog, attract freelance clients, and be more like what I truly wanted, rather than being heavily based on Pinterest posts and YouTube videos I had watched for inspiration.

## The Tech Stack

I built this site with a modern front-end stack:

- `Vue 3` — component-based UI with the Composition API
- `Vite` — fast dev server and build tooling
- `Vuetify` + `Tailwind CSS` — components and utility-first styling
- `Vue Router` — client-side routing between pages
- `markdown-it` — rendering these blog posts from Markdown
- `GitHub Pages` — free, simple hosting straight from the repo

The biggest changes were of course the modern UI framework additions and adding the usage ot markdown-it to render these blog posts. I'm honestly not sure how most people go about doing this themselves since there are so many options, but I'm happy with it for the time being!

## Design & Aesthetic

Despite the hate that many people feel towards frosted glass in UIs...I'm honestly a huge fan! Ever since Apple introduced it, I immediately fell in love with the futuristic feel that it can give a UI. As for the space theme, I think space is a beautiful scenery (despite not having been there lol) and I thought it would be a lot of fun to design my page around it. I wanted the page to feel reactive, yet not create something too distracting. That being said, my YouTube algorithm has been showing me some of the amazing visuals that can be created with `Three.js` and I'm definitely intrigued. Maybe in the near future I can experiment with it in my free time!

## Challenges & Lessons Learned

I definitely had to take some breaks between refactors to give myself time to think about how I wanted the site to look/operate. As I previously said, this also would not have been done without Claude since it would have taken FOREVER! AI has truly been a gamechanger (maybe I'll write my own synopsis on this in the near future like everyone else has), but I've been loving using it in both my free time and at work.
